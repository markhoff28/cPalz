## Switch Statements
Beispiel in Java:
```java
int day = 4;
switch (day) {
  case 6:
    System.out.println("Today is Saturday");
    break;
  case 7:
    System.out.println("Today is Sunday");
    break;
  default:
    System.out.println("Looking forward to the Weekend");
}
```
Umsetzung in cPalz:
```java

  option 6:
    schreib("Samsdaach");
    uffhere;
  option 7:
    schreib("Sundach");
    uffhere;
  schunschd:
    schreib("Looking forward to the Weekend");
}
```
